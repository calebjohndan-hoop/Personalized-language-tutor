# Personalized Language Tutor (OpenAI API Project)

## Overview
This project is an AI-powered personalized language tutor that helps users improve language skills using speech processing and advanced language models.
It uses OpenAI APIs to handle speech-to-text conversion, translation, grammar correction, and pronunciation feedback.

## Features
- 🎤 Speech-to-text transcription using Whisper-1
- 🌍 Translation of transcribed text using GPT-4o-mini
- ✍️ Grammar analysis and correction of translated text
- 🗣️ Pronunciation feedback by comparing original and target text
